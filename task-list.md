# 开发规则

## AI 开发原则

1. 一次只执行当前阶段任务
2. 完成后必须测试
3. 不提前开发未开始阶段
4. 不修改已经完成模块，除非发现Bug
5. 每完成一个 Phase 创建 Git commit


## 完成定义

每个任务必须满足：

- 功能可运行
- 手机端正常显示
- 无明显报错
- 数据真实写入 Supabase
