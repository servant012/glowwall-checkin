## 技术决策记录

| 决策 | 选择 | 原因 |
|------|------|------|
| 部署平台 | Vercel（非 Cloudflare Pages） | 有 MCP 工具可用，一键部署 |
| Supabase SDK 加载 | jsdelivr CDN，同步加载 | 最初放在 head 阻塞渲染，后改为 body 底部 |
| config.js 管理 | gitignore + Vercel 构建时从环境变量生成 | 不提交密钥到 git |
| 图片压缩 | Canvas 二分法压缩，目标 500KB | 减少存储和加载时间 |
| 连续天数算法 | 从今天回溯，缺一天即断 | 简单可靠 |
| 时区处理 | UTC+8 硬编码偏移 | 目标用户全在中国 |
| 瀑布流 | CSS column-count: 2 | 简单，无需 JS 布局库 |

5. 如果有 JS 报错 → 根据报错定位修复
